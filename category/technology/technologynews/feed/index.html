]]> January 14, 2020, is the date marked by Microsoft to end it’s support for Windows 7, thus hoping to push everyone to Windows 10. While this may seem daunting to a few, there are several options you can take. First and foremost, you can follow the recommendations set forth by Microsoft here, However these seem a bit drastic (making you buy a new PC from the start).

First check your computer system and see if it can support Windows 10.

  • Processor: 1 gigahertz (GHz) or faster processor or SoC.
  • RAM: 1 gigabyte (GB) for 32-bit or 2 GB for 64-bit.
  • Hard disk space: 16 GB for 32-bit OS 20 GB for 64-bit OS.

If it does support it, then great, you can upgrade to Windows 10 to keep getting updates.
But if it doesn’t , this doesn’t mean you need to quickly upgrade to Windows 10 before January 14 or else your computer will fail to operate. This is simply the last date for updating your system. Ensure your system is up to date and your data is backed up. From there, do consider purchasing a new computer within the next few months (unless you want to switch to Linux, which I will say is a fun route for the adventurous). If you don’t however, don’t expect your system to be secure (try running XP, finding updates and seeing what kind of vulnerabilities exist today for it).
